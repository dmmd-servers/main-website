# Main Website

## Synopsis

This is my portfolio website.

## Installation

```
# Clone the repository
git clone https://github.com/dmmd-servers/main-website
cd main-website

# Install packages
bun i

# Starts server
bun .
```

## Configuration

```.env
# Server port.
PORT=3000
```
